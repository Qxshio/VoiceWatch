Supported browsers

          
The bundled unpacked setup supports Chromium-based browsers with native messaging, including Chrome, Edge, Brave, Vivaldi, Opera, and Chromium. Firefox uses the separate Firefox package attached to each Voice Watch release.

          
Because Voice Watch is open source and privacy-first, you load the released connector folder yourself. That keeps the browser permission step visible, lets anyone inspect what is being installed, and avoids a hidden store package touching your Roblox session without you choosing it.
